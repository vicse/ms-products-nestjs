## Description

Microservicio de productos NestJS, PostgresQL, Prisma

## Project setup

1. Clonar el proyecto
2. Crear un archivo `.env` basado en el archivo `.env.template`
3. Instalar las dependencias del proyecto

```bash
$ npm install
```
4. Levantar el proyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
